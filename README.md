# HoloLens2 Demo - Augmented Reality 



<!-- ABOUT THE PROJECT -->
## About The Project

HoloLens 2 ([HoloLens 2](https://www.microsoft.com/en-us/hololens/buy))

Get communication between HoloLens 2 and a local server, and should be able to extract values from the server with POST and GET request and store them in a SQLite database consisting of DLL library. 


### Built With
Universal Windows Platform (UWP) that is apart of .NET framework and it is used to create client applications for Windows. UWP apps use WinRT APIs to provide powerful UI and advanced asynchronous features that are ideal for internet-connected devices.
<p align="right">(<a href="#readme-top">back to top</a>)</p>




### Prerequisites


1. HoloLens 2 
2. Windows 10
3. NET Framework version 4.
4. Visual Studio 2019/2022
5. Visual Studio UWP Tool, to build and compile the application

<h4>It is also possible to test application without HoloLens 2 by connecting application to a virtual emulator (Hyper-V), this one must have a Windows 10 version that supports Hyper-V function. </h4>
